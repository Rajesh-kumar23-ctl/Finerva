# Finerva - AI-Powered Financial Intelligence Platform

## Project Overview
**Finerva** is a comprehensive, web-based financial education and trading intelligence platform powered by AI. It provides users with real-time market insights, portfolio management tools, interactive learning modules, and AI-driven advisory services.

---

## Key Features

### 1. Login/Registration System
- Clean, modern authentication interface
- Smooth tab-switching between Login and Register
- Redirects to main dashboard upon successful login
- Floating card animation with hover effects

### 2. Stock Market Predictor
- AI-powered market momentum simulation
- Interactive stock ticker input (TSLA, NVDA, AAPL, etc.)
- Visual chart representation using Chart.js
- Profit/Loss calculation based on user predictions
- Button hover animations and glow effects

### 3. AI Strategy Advisor
- 360-degree Financial AI Chatbot
- Context-aware responses about:
  - Tax optimization
  - Inflation protection
  - Real estate vs stocks
  - Retirement planning
  - Portfolio diversification
- Quick action pills with hover animations
- Chat message slide-in animations

### 4. News Intelligence
- Live news feed with real-time updates
- Dynamic news generation across categories (Stocks, Crypto, Commodities, Economy)
- Breaking news popup alerts
- Auto-refresh every 20 seconds with countdown timer
- Digital clock display
- Card hover animations with staggered delays

### 5. Portfolio Analyzer
- Add and track investment assets
- Real-time P/L calculations
- Table row hover effects
- Input focus animations with glow effects
- Floating card animation

### 6. Professional Academy (Learning Module)
- 10 comprehensive topics x 10 modules = 100 elite training modules
- Topics include:
  - Market Fundamentals
  - Technical Analysis
  - Fundamental Analysis
  - Risk Management
  - Derivatives
  - Fixed Income
  - Crypto Assets
  - Behavioral Finance
  - Portfolio Strategy
  - Macro Economics
- 3-Phase learning system:
  1. Definition/Explanation
  2. Key Takeaways Summary
  3. Interactive MCQ Assessment with feedback
- Sidebar slide-in animation
- Topic hover effects with scale and shadow
- Quiz option animations (correct pulse, wrong shake)
