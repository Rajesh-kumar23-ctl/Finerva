# Finerva - AI Financial Intelligence Platform

Finerva is an AI-powered financial platform offering stock prediction, portfolio management, real-time market news, and interactive learning modules for trading education.

## Features

### 1. **Authentication System**
- Secure user registration and login
- JWT-based authentication
- Session management

### 2. **Dashboard**
- User progress tracking
- Learning module completion status
- Quick access to all features
- Real-time notifications

### 3. **AI Stock Predictor**
- Stock momentum prediction
- Interactive chart visualization
- AI-powered analysis
- Profit/Loss simulation

### 4. **Portfolio Analyzer**
- Add and track investments
- Real-time P/L calculation
- Portfolio diversity overview
- Visual data representation

### 5. **News Intelligence**
- Real-time market news
- AI-powered sentiment analysis
- Impact score indicators
- Breaking news notifications

### 6. **AI Financial Advisor**
- Interactive chatbot
- Personalized financial advice
- Portfolio-based recommendations
- Strategic query responses

### 7. **Learning Modules**
- 20 comprehensive modules
- Interactive quizzes
- Progress tracking
- Score-based completion

## 🛠️ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   
```
bash
   git clone <repository-url>
   cd "gift hkt"
   
```

2. **Install dependencies**
   
```
bash
   npm install
   
```

3. **Start the server**
   
```
bash
   node server.js
   
```

4. **Open in browser**
   
```
   http://localhost:3000
   
```

## 🔌 API Endpoints

### Authentication

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/register` | Register new user |
| POST | `/api/login` | User login |
| GET | `/api/user` | Get user data |

### Progress Tracking

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/progress` | Get user progress |
| POST | `/api/progress` | Update progress |

### Web3 (Optional)

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/web3-login` | Login with MetaMask |
| POST | `/api/web3-register` | Register with MetaMask |
